Ferticulture

Ferticulture predicts precise NPK values for soil with 91% accuracy using a Kaggle dataset. The backend employs Python (numpy, matplotlib, seaborn, scikit-learn) with RandomForest regressors and SVMs, while the frontend uses Kivy for a seamless user experience. This project enhances soil fertility management for optimal agricultural productivity.

Supermarket Management Database System

I developed a Supermarket Management System to streamline inventory and transaction management for shopkeepers. Using Python for the frontend and MySQL for the backend, this system efficiently tracks item availability, automates record-keeping, and provides real-time updates, reducing manual work and enhancing overall efficiency.

ADAC

This project develops an AI-based adaptive cruise control and lane departure warning system. Using TensorFlow, Keras, and OpenCV, the system employs LSTM models for cruise control and computer vision techniques for lane detection, ensuring safe and efficient driving by maintaining optimal speed and preventing unintentional lane departures.
